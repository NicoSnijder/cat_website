Welkom!

Deze website heb ik geschreven om als soort hub te functioneren tijdens Catechisatie. Hierop vind je een korte versie van de verschillende lessen die ik heb voorbereid.

Er is ook een pagina waar je gebedspunten kunt achterlaten. Als ik in de komende weken ergens voor moet bidden, kun je het anoniem achterlaten. Dan neem ik het mee in mijn gebed.

Deze website is gemaakt met behulp van Markdown, HTML en een variatie op Simple.css met dank aan Kev Quirk. En de gastheer van de website is Netlify.
